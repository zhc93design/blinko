macOS desktop signal light

Winko

把 AI Agent 的沉默，翻译成一盏会呼吸的灯。
Turn AI agents' quiet work into a small, expressive signal light.

Winko watches local agent activity across CatDesk, Claude Code, Codex, Cursor, OpenClaw, Z Code, and OpenCode, then turns it into one calm desktop signal.

Download for macOS View releases

Winko is for people who work with multiple AI coding agents. It turns scattered logs, hooks, files, and local status streams into a simple ambient signal: green when everything is calm, yellow when your attention would help, red when something needs a decision or failed.

Winko 是给多 AI Agent 工作流准备的桌面陪伴工具。它把终端、编辑器、日志和本地状态流里分散的事件，整理成一眼能读懂的灯语：绿色表示安心，黄色表示值得看一眼，红色表示需要处理。

Signal Language / 灯语

idle

空闲或完成。Nothing active, all clear.

working

正在思考、执行工具或编辑。Thinking or working.

attention

可能需要输入或关注。Needs your eyes.

error

权限、确认、失败或阻塞。Decision or failure.

silent

手动关闭或静默。Muted/off.

Privacy Shape / 隐私边界

Winko is designed as a local status translator. It watches local hooks, local logs, local files, or local server status streams. For read-only integrations, it uses structured status metadata and avoids reading prompt or model message bodies for signal decisions.

Winko 的定位是本地状态翻译器。它监听本地 Hook、本地日志、本地会话文件或本机状态流；对于只读集成，Winko 使用结构化状态元数据判断灯态，避免读取 prompt 或模型正文。
