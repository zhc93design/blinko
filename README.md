# Blinko

Blinko is a macOS floating signal light for AI agent work status.

It watches local agent activity from CatDesk, Claude Code, Codex, Cursor, OpenClaw, Z Code, and OpenCode, then turns that activity into a small desktop light so you can see whether an agent is working, waiting, blocked, or done.

## Download

Download the latest macOS build from:

https://github.com/zhc93design/blinko/releases/latest

## Updates

Blinko checks this public release repository for signed and notarized macOS updates. The source code is developed separately in a private repository; this repository is the public download and update channel.
